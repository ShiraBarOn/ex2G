Walkthrough:
1. set Health to 200.
2. set Jumpheight to 30
3. Jump over to upper level
4. get down and set the Shooting Angle to 0
5. Shoot the enemy
6. set acceleration to 30
7. go over the spikes
8. set acceleration to 7 and Jump height to 20
9. jump to the next level.
10. set jump height to 30 and jump and shoot the closest enemy.
11. jump to the next level and kill the second enemy.
12. fall down and go to the yellow square.
13. You win! Yay :)
